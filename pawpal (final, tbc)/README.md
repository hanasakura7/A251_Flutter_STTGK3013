# PawPal: Final Assessment

A continuous full-stack app on pet management (adoption, medical, donate)

Google Drive (Database screenshot and App Screenshots): https://drive.google.com/drive/folders/1NyyB490NdYBVYi2EfBUhjndfjkr4ghyt

## Project Setup

Front-end: Flutter (Mobile)

Back-end: PHPMyAdmin 

Database: MySQL

## App Features (Some might not be fully functional)

- Register

- Log-in 

- Main Screen (Showing user's name which are registered in database) for 3 seconds, then shows to the list of pets by the owner

- Submit Pet feature (Up to 3 pics, 1st pic for thumbnail)

- Pets Uploaded by User

- Pet Lists (Uploaded by all users)

- Adoption Request (made by users)

- Donations to animal

- Sidebar: Clickable and leads to Home, List of Animals, Donation History, Uploaded Pets, Profile, Settings, and Logout (Redirect to Login Page)
